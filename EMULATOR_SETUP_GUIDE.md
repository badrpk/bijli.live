# 🚀 BIJLI.LIVE - BROWSER VIEWING SETUP GUIDE

## 📱 **ANDROID STUDIO EMULATOR SETUP**

### **Step 1: Open Android Studio**
```
1. Press Windows Key + R
2. Type: "C:\Program Files\Android\Android Studio\bin\studio64.exe"
3. Press Enter
4. Android Studio will open
```

### **Step 2: Open Your Project**
```
1. In Android Studio: File → Open
2. Navigate to: D:\bijli.live
3. Click "OK"
4. Wait for project to load and sync
```

### **Step 3: Create Virtual Device (AVD)**
```
1. In Android Studio: Tools → AVD Manager
2. Click "Create Virtual Device"
3. Choose "Phone" category
4. Select "Pixel 6" (recommended)
5. Click "Next"
6. Choose "API 34" (Android 14) or latest
7. Click "Next"
8. Click "Finish"
```

### **Step 4: Run Your App**
```
1. In Android Studio: Click "Run" button (green play icon)
2. Select your virtual device
3. Click "OK"
4. Wait for emulator to start
5. Your bijli.live app will launch in emulator
```

## 🔧 **TROUBLESHOOTING**

### **Common Issues**

#### **Issue 1: Emulator Won't Start**
```
Solution:
1. Enable Virtualization in BIOS
2. Install Intel HAXM or AMD Hypervisor
3. Check Windows Hyper-V settings
```

#### **Issue 2: Project Won't Build**
```
Solution:
1. File → Sync Project with Gradle Files
2. Build → Clean Project
3. Build → Rebuild Project
```

#### **Issue 3: Database Connection Issues**
```
Solution:
1. Check PostgreSQL is running
2. Verify password: Karachi5846$
3. Check localhost:5432 connection
```
