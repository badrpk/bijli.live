## 📱 **STEP-BY-STEP MOBILE PHONE INSTRUCTIONS**

### **Method 1: Direct Android Studio Installation (Recommended)**

#### **Step 1: Open Project**
```
1. Open Android Studio
2. File → Open → Select: D:\bijli.live
3. Wait for Gradle sync to complete
```

#### **Step 2: Connect Your Phone**
```
1. Connect your Android phone to computer via USB cable
2. Enable USB debugging on your phone:
   - Settings → About Phone → Tap "Build Number" 7 times
   - Settings → Developer Options → Enable "USB Debugging"
3. Allow USB debugging when prompted on phone
```

#### **Step 3: Install and Run**
```
1. In Android Studio, click "Run" button (green play button)
2. Select your connected phone from device list
3. Click "OK" to install and run
4. App will be automatically installed and launched on your phone
```

### **Method 2: Manual APK Installation**

#### **Step 1: Build APK**
```
1. In Android Studio: Build → Build Bundle(s) / APK(s) → Build APK(s)
2. APK will be created at: D:\bijli.live\app\build\outputs\apk\debug\app-debug.apk
```

#### **Step 2: Transfer to Phone**
```
1. Copy app-debug.apk to your phone
2. Enable "Install unknown apps" in phone settings
3. Tap the APK file on your phone to install
4. Launch the app from your phone's app drawer
```

## 🔧 **TROUBLESHOOTING**

### **Phone Not Detected**
```
Solutions:
1. Check USB cable connection
2. Enable USB debugging
3. Install phone drivers
4. Try different USB port
5. Restart Android Studio
```

### **APK Installation Failed**
```
Solutions:
1. Check phone storage space
2. Enable "Install unknown apps"
3. Uninstall previous version
4. Check Android version (7.0+)
5. Enable "Allow from this source"
```

### **App Crashes on Mobile**
```
Solutions:
1. Check database connection
2. Verify PostgreSQL is running
3. Check app permissions
4. Review Android Studio logs
5. Ensure network connectivity
```
